Welcome to the Zero-Code Demo Site

A fully client-side, multi-page demo using only HTML, CSS and JavaScript.

Zero backend

Everything runs in the browser. Authentication is simulated using localStorage.

Admin & User

Two separate dashboards with role-guarding and simulated data.

Responsive

Simple responsive layout for desktop and mobile screens.

How it works

  1. On this home page choose a role (Admin/User).
  2. The site stores the chosen role in localStorage and opens the matching dashboard.
  3. Dashboards check role and will redirect to Home if not permitted.
